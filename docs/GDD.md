# 摸鱼大师 - 微信小游戏完整策划案

## 一、游戏概述

- **类型**：点击叠放三消（核心框架复刻"羊了个羊"）
- **主题**：职场摸鱼文化
- **操作**：单指点击
- **美术风格**：纯文字卡片 + 极简UI，零贴图成本
- **核心循环**：点击卡片 → 进入槽位 → 3张同卡消除 → 触发肉鸽技能 → 局外成长
- **变现模式**：纯广告变现（激励视频），个人主体可上线

## 二、核心玩法

### 2.1 双重限制机制

步数和卡槽是两套**并行的限制机制**，互相独立但共同制造紧张感。不能直接删除一个，但可以通过技能和道具临时突破。

| 机制 | 说明 | 失败条件 |
|------|------|----------|
| **步数** | 控制你能点击的总次数（如第二关35步），每次点击消耗1步 | 步数耗尽 → 失败 |
| **卡槽** | 控制你能同时持有几张卡（7格），卡入槽后占据一格 | 槽满7张且无法消除 → 被老板发现，失败 |

**两者是"双重死亡线"：**

| 状态 | 体验 |
|------|------|
| 步数足、槽位足 | 顺利推进 |
| 步数足、槽位满 | 必须立刻消除解围 |
| 步数少、槽位足 | 必须精打细算每次点击 |
| 步数少、槽位满 | 绝境，只能靠技能救命 |

**为什么两者必须并存：**
- 如果删除步数（只有槽位限制）→ 变成纯"耐心排序"，没有行动压力，紧张感大幅下降
- 如果删除槽位（只有步数限制）→ 变成"无限容量的连连看"，失去空间管理策略，太简单
- 两者并存才是"差一步就过"的上头核心

**但可以"软性删除"——通过肉鸽技能临时突破：**

| 突破方向 | 技能/道具 |
|----------|-----------|
| 暂时无视槽位上限 | S1「飞行模式」：接下来3次点击不占槽位 |
| | S13「装死模式」：接下来5步槽满上限临时变为9 |
| | S2「同事掩护」：本局槽位上限+1 |
| 暂时无视步数压力 | S7「极限逃生」：额外+5步 |
| 传说级"删除限制" | 🦄「忘记打卡」：本局取消步数限制，但槽位上限-2 |
| | 🔓「无人办公室」：本局槽位上限取消，但步数减半 |

### 2.2 基础规则

- 屏幕上方堆叠大量卡片，每张卡展示一种"摸鱼行为"
- 屏幕下方有7个槽位（工位风险值）
- 玩家点击**未被压住的卡片**，卡片自动飞入槽位，消耗1步
- 槽位中集齐3张相同卡片 → 自动消除，获得快乐值
- **步数耗尽**或**槽位满7张且无法消除** → 游戏失败
- 棋盘清空 → 通关成功

### 2.3 关卡设计

| 关卡 | 卡片种类 | 堆叠层数 | 初始步数 | 通关率 |
|------|----------|----------|----------|--------|
| 第一关 | 3-4种 + 1-2种轻度负面卡 | 1-2层 | 充裕（约30步） | ~100% |
| 第二关 | 6-8种+全部功能卡 | 3-5层 | 紧张（约35步） | 3%-8% |

### 3.1 基础摸鱼卡（7种）

| 图标 | 卡名 | 稀有度 | 出现频率 |
|------|------|--------|----------|
| 📱 | 刷手机 | 普通 | 极高 |
| 🚽 | 带薪拉屎 | 普通 | 高 |

| 🎮 | 偷偷游戏 | 稀有 | 低 |

### 3.2 ❓事件卡机制

所有功能卡（正面/负面/双刃剑）在棋盘上**统一显示为❓事件卡**，玩家点击后才能揭示真实身份。

**设计意图：**
- 如果负面卡明牌显示，玩家会直接跳过不点，负面卡形同虚设
- 统一为❓后，玩家面临"赌或不赌"的决策，产生紧张感
- 必须点的场景：❓事件卡压在你需要的卡上面，不点就拿不到 → 被迫冒险
- 不确定的场景：❓旁边有安全普通卡 → 选安全还是赌？策略空间打开
- 抽到好卡爽感翻倍（"居然是带薪年假！"），抽到坏卡有心理缓冲（"我就知道……"）

**保底机制：**
> 连续3张❓事件卡揭示为负面卡后，第4张❓事件卡**自动揭示为正面卡**，防止连续踩雷导致崩溃退出。

**与已有系统的联动：**

| 已有设计 | 联动效果 |
|----------|----------|
| 🍀 领导的宠儿 | 下3次点击不触发负面 → 赌❓的黄金窗口，技能价值暴涨 |
| 🍵 茶水间情报 | 揭示隐藏卡片 → 同时揭示❓的正面/负面身份，信息型技能变强 |
| 负面卡常在顶层 | ❓压在你需要的卡上面，不点就拿不到 → 被迫冒险 |
| 正面卡压在2层以下 | 被普通卡/❓盖住，更难获取 |

#### 正面卡

| 卡片 | 效果 | 概率 |
|------|------|------|

#### 双刃剑卡

| 卡片 | 效果 | 概率 |
|------|------|------|
| 🔀 岗位轮换 | 槽中一张卡随机变成另一种卡 | 中 |
| ⏳ 加班申请 | 步数+5，但所有"老板巡视"卡浮到顶层 | 低 |
| 🍺 部门聚餐 | 随机移除3张卡，但槽位上限临时-1 | 低 |
| 📊 全员周报 | 揭示所有隐藏卡位置，但牌堆增加2张"老板巡视" | 低 |

- 功能卡**第一关**轻度出现1-2种负面卡（"老板巡视"+"系统崩溃"），让玩家提前理解❓事件卡机制；**第二关**全部功能卡登场，负面约30%，正面约15%，双刃剑约15%

- 负面❓常放在顶层（压住你需要的卡，迫使你冒险），正面❓多压在2层以下（更难获取）
