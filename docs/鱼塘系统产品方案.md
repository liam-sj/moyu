# 摸了个鱼 — 鱼塘系统完整产品方案

> 参考羊了个羊"羊群"机制，结合职场摸鱼主题，设计12鱼塘社交排行体系。
> 核心差异：羊了个羊用"省份"自动分组，我们用"选鱼"主动选择——选鱼即选鱼塘+选职业，一步到位两层身份。

---

## 一、系统概述

### 1.1 一句话定位

**通关加入鱼塘，为鱼塘而战，为职业争光。**

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 归属感 | 给玩家一个"我是XX塘的人"的身份认同，每天上线有归属 |
| 竞争欲 | 鱼塘排行制造"不能给鱼塘拖后腿"的压力 |
| 传播力 | 选鱼/为鱼塘贡献/鱼塘霸榜 → 天然社交货币 |
| 留存钩 | 排行每日重置 + 人均排行 + 霸榜奖励 → 日日有理由回来 |

### 1.3 与羊了个羊的对比

| 维度 | 羊了个羊（羊群） | 摸了个鱼（鱼塘） |
|------|-------------------|-------------------|
| 分组方式 | 省份自动归属（基于定位） | 12条鱼主动选择 |
| 分组数量 | 34个（省级行政区） | 12个鱼塘 |
| 身份深度 | 只有一层（省份） | 两层（鱼塘+职业暗喻） |
| 加入条件 | 通关当天关卡 | 首次通关后选鱼加入 |
| 排行维度 | 省份总羊数 | 今日最肥/人均摸鱼王/连续霸榜 |
| 社交表达 | 十字架墓地（失败次数） | 鱼塘标语+职业梗+贡献排名 |
| 更换机制 | 无法更换（出生地决定） | 每周可换1次 |
| 情绪钩子 | "为家乡而战" | "为鱼塘争光"+"我是什么鱼" |

### 2.2 设计原则

- **标语即梗**：不需要理解职业暗喻也能玩，理解了的会心一笑
- **自嘲不攻击**：所有标语都是自嘲式幽默，不贬低任何职业
- **审核安全**：表面就是"选鱼"，不涉及真实职业信息的收集和展示
- **色彩区分**：12个鱼塘12种颜色，排行榜和UI上一眼辨识

## 四、选鱼流程（核心入口）

### 4.1 触发时机

**首次通关第二关后**弹出选鱼界面。

> 为什么不是第一关？第一关是教学关，太容易通关，选鱼的仪式感不足。第二关通关才有"我配选鱼"的荣誉感。参考羊了个羊——只有通关者才配加入羊群。

### 4.3 交互细节

| 交互 | 说明 |
|------|------|
| 点击鱼卡片 | 弹出确认弹窗："选择小金鱼🐟，加入摸鱼塘？" |
| 确认选择 | 播放"入水"动画：鱼跳入对应鱼塘，水花溅起 |
| 选择后 | 自动返回首页，首页"我的鱼塘"卡片更新 |
| 首次选择 | 不可跳过，必须选一条鱼才能继续 |
| 再次选择 | 每周可换1次，换鱼时弹出提示："换鱼后鱼塘贡献归零，确定？" |

### 4.4 鱼卡片信息

每张鱼卡片包含：

| 元素 | 说明 |
|------|------|
| 鱼emoji | 大号居中，有轻微浮动动画 |
| 鱼塘名 | 卡片下方 |
| 标语 | 鱼塘名下方，小字灰色 |
| 当前鱼数 | 角标显示"今日XX条鱼" |
| 主题色边框 | 卡片边框使用鱼塘主题色 |

### 4.5 换鱼流程

```
首页"我的鱼塘"卡片 → 点击"换鱼"按钮
  → 弹出12鱼塘选择界面
  → 点击新鱼 → 确认弹窗（"换鱼后鱼塘贡献归零"）
  → 确认 → 入水动画 → 更新鱼塘
```

**限制规则：**
- 每周可免费换1次
- 换鱼后原鱼塘贡献清零（个人数据如经验、等级保留）
- 新鱼塘贡献从0开始累计
- 换鱼有7天冷却期

### 5.3 三个排行维度

#### 🏆 今日最肥鱼塘

| 规则 | 说明 |
|------|------|
| 计算方式 | 鱼塘内所有玩家今日通关数总和 |
| 更新频率 | 实时 |
| 重置规则 | 每日0点清零，新一天重新计数 |
| 展示 | Top3大卡片 + 4-12名列表 |

#### 👑 人均摸鱼王

| 规则 | 说明 |
|------|------|
| 计算方式 | 今日通关总数 ÷ 鱼塘活跃人数 |
| 更新频率 | 每小时更新 |
| 设计意图 | 解决人数不均问题（程序员塘人最多，但人均可能不如小塘） |
| 展示 | 同上，Tab切换 |

#### 🔥 连续霸榜

| 规则 | 说明 |
|------|------|
| 计算方式 | 连续天数占据"今日最肥"第一名的鱼塘 |
| 更新频率 | 每日0点结算 |
| 展示 | 显示连续天数 + 历史最长霸榜记录 |
